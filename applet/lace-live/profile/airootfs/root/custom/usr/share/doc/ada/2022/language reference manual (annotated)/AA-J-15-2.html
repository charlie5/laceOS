J.15.2 Pragma No_Return

Syntax

1/3
{AI05-0229-1} The form of a pragma No_Return, which is a representation pragma (see 13.1), is as follows:
2/5
{AI12-0269-1}   pragma No_Return (subprogram_local_name procedure_local_name{, subprogram_local_name procedure_local_name});

Legality Rules

3/5
{AI05-0229-1} {AI12-0269-1} Each subprogram_local_name procedure_local_name shall denote one or more subprograms procedures or generic subprograms procedures. [The subprogram_local_name procedure_local_name shall not denote a null procedure nor an instance of a generic unit.]

Static Semantics

4/5
{AI05-0229-1} {AI12-0269-1} Pragma No_Return specifies that the No_Return aspect (see 6.5.1) for each subprogram procedure denoted by each local_name given in the pragma has the value True. 

Wording Changes from Ada 2005

4.a/3
{AI05-0229-1} {AI05-0299-1} This subclause is new. Pragma No_Return was moved here from 6.5.1; aspect No_Return lives there now. 

Wording Changes from Ada 2012
